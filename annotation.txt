1. Файл 'http_methods' - содержит кастомные методы для отправки запросов
2. Файл 'api' - для хранения методов с помощью которых будем тестировать google maps api
3. Файл 'test_google_maps_api.py' - для хранения тестов google maps api
4. Файл 'checking' - хранит в себе методы для проверки ответов наших запросов
5. Файл 'test_checking_google_maps.py' - для хранения тестов google maps api с проверкой статус кода,
 наличие обязательных полей, значений полей
6. Файл 'darth_vader.py' - для получения списка всех персонажей игравших в фильмах
 звездных войнов вместе с Дарт Вейдером
7. Файл 'hero_names' - для хранения имен наших героев из фильмов звездые войны
8. Файл 'logger.py' - хранит в себе класс для записи логов
